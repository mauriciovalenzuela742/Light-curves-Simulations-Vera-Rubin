# SIMLIB - Análisis de Librerías de Observaciones

## Descripción

Esta carpeta contiene notebooks y análisis de las librerías de observaciones (SIMLIB) utilizadas en las simulaciones de SNANA para Vera Rubin LSST.

## Contenido

### Notebooks de análisis
Los notebooks en esta carpeta analizan estadísticas y características de las librerías SIMLIB:
- Distribución de observaciones
- Cobertura por filtro  
- Profundidad fotométrica
- Patrones temporales
- Comparación entre versiones
